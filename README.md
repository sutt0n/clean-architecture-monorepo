# Overview

This is a boilerplate repository for a Lerna monorepo implemented via clean architecture.

## Requirements
- Lerna: `npm i -g lerna`

## Setup
1. `yarn`
2. `lerna bootstrap`
3. `yarn start`

## Tests
- `yarn test`
